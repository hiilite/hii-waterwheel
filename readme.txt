== Description ==

This plugin allows you create a basic waterwheel image carousel

== Installation ==

Installing "HiiWaterwheel" can be done by using the following steps:
1. Download the plugin after purchase
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
== Frequently Asked Questions ==
= What does this plugin do? =
This plugin allows you create a basic waterwheel image carousel.
= How do I create a waterwheel? =
1. Go to HiiWaterwheels > Add New in the left hand menu
2. Give your waterwheel a title. (This will not display anywhere on the site and is just for the purpose of identifying your waterwheel in the dashboard)
4. Click Add Image and select you image from the Media gallery
5. Use the radio buttons to set you image to either Cover or Contain
6. Click Add Another Slide to add the next image 
7. Click Publish when finished
= How do I add a waterwheel to my page? =
1. Go to HiiWaterwheel > All Waterwheels
2. Copy the shortcode for your waterwheel
3. Go to the page where you want to display the waterwheel
4. Paste the shortcode into the page where you want to display it
5. Click Update
= What are the different waterwheel side setting and what do they do? =
Add Image: Select an image from the Media gallery or upload a new image.

Image Display Contain: This will fit the image to the smallest dimension of the container and scale the image proportionally. This will not crop the image but may show the slide background color.

Image Display Cover: This will scale the image up proportionally to fill the slide. This will not show any of the slide background color but may crop the image.

Up Arrow: Moves the slide up in the slide order.

Down Arrow: Moves the slide down in the slide order.

Remove Slide: Deletes the slide from the waterwheel.

Add Another Slide: Adds a new slide to the waterwheel.
